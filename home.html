Frequently asked questions

Answers to the topics teams ask us before going live.

Can I add my own conversion recipes?
Absolutely. Extend the conversion map in main.js with custom validation rules, timing, and messaging.
How do I swap the simulated link for a real download?
Submit your file to an API endpoint, then replace the placeholder href in buildSuccessState with the signed URL you return.
Does FOLNEB support multiple uploads?
Yes. Recipes like Merge PDFs allow multiple file selection and adapt the helper text to keep users on track.
Can we localize the interface?
The structure is semantic and ready for translation strings. Update the copy in each template or load localization files dynamically.
